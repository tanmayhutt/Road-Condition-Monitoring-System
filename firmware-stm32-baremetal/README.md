STM32F446RE bare-metal firmware for the road surface monitoring system.



\- Bare-metal drivers for LSM6DS3TR-C IMU (I2C), HC-SR04 (TIM5 input capture + GPIO),

&#x20; and UART links to GPS (NEO-6M) and ESP32.

\- Non-blocking sensor fusion loop running at 100 Hz.

\- Built with STM32CubeIDE targeting Nucleo-F446RE.
